## Pattern Overview

Chronic vomiting combined with weight loss in cats is a common but important presentation that demands structured workup. The leading differentials are hyperthyroidism (especially cats > 8 years old), inflammatory bowel disease (IBD), small cell lymphoma of the GI tract, chronic kidney disease, and pancreatitis. In cats, these diseases frequently co-exist (the "triaditis" of IBD + pancreatitis + cholangiohepatitis). A methodical approach — starting with minimum database and T4 — guides next steps efficiently.

This document supports AI-assisted triage and test recommendation. It does not constitute a diagnosis.

## Common Symptoms

- Intermittent or chronic vomiting (food, bile, or hairballs)
- Progressive weight loss over weeks to months
- Muscle wasting despite maintained or increased appetite (hyperthyroidism)
- Decreased appetite (CKD, lymphoma, IBD)
- Diarrhea (may be intermittent or chronic)
- Increased thirst and urination (hyperthyroidism, diabetes, CKD)
- Palpable thickened intestinal loops (IBD, lymphoma)
- Palpable thyroid nodule in neck (hyperthyroidism)

## Possible Associated Diseases

| Disease / Pattern                  | Key Differentiating Feature                                                      |
| ---------------------------------- | -------------------------------------------------------------------------------- |
| Hyperthyroidism                    | Cats > 8 years; weight loss + polyphagia; elevated T4; palpable thyroid nodule   |
| IBD (inflammatory bowel disease)   | Chronic course; normal or mildly elevated labs; GI biopsy diagnostic             |
| Small cell GI lymphoma             | Similar presentation to IBD; definitive via endoscopic biopsy or surgical biopsy |
| Chronic kidney disease (CKD)       | Older cat; PU/PD; elevated CREA + BUN; low urine SG                              |
| Pancreatitis                       | Subtle in cats; elevated fPLI; abdominal ultrasound changes                      |
| Cholangiohepatitis / liver disease | Elevated ALT + ALP + GGT; icterus; common in cats                                |
| Feline triaditis                   | IBD + pancreatitis + cholangiohepatitis — concurrent in many cats                |
| Diabetes mellitus                  | PU/PD; hyperglycemia; glucosuria; weight loss despite polyphagia                 |
| GI lymphoma (large cell)           | Rapid deterioration; palpable mass; hypoechoic lesions on ultrasound             |

## Recommended Diagnostic Tests

**First-line:**

- **CBC with differential** — anemia pattern, lymphocyte count
- **Comprehensive serum chemistry panel + T4** — T4 is mandatory in cats > 8 years; chemistry for organ involvement
- **Urinalysis with USG** — CKD assessment, glucosuria (diabetes)

**Second-line:**

- **Feline pancreatic lipase (fPLI / Spec fPL)** — pancreatitis; often concurrent with IBD
- **Abdominal ultrasound** — intestinal wall thickness and layering (IBD vs. lymphoma), liver, pancreas, kidneys, lymph nodes
- **FeLV/FIV testing** — FeLV-associated lymphoma
- **Serum B12 (cobalamin)** — low in severe small intestinal disease; guides prognosis and supplementation

**If IBD vs. lymphoma differentiation needed:**

- **GI endoscopy + intestinal biopsy** — definitive for IBD vs. small cell lymphoma; preferred first approach
- **Full-thickness surgical biopsy** — if endoscopic biopsy inconclusive

## When to Prioritize Urgent Evaluation

- Rapid weight loss (> 10% body weight over 2–4 weeks)
- Complete anorexia lasting > 48–72 hours (hepatic lipidosis risk if overweight)
- Icterus (yellow eyes, skin, gums) — liver failure or hemolysis
- Severe dehydration or collapse
- Palpable abdominal mass — neoplasia until proven otherwise
- Diabetic ketoacidosis signs: vomiting + lethargy + acetone breath + hyperglycemia
