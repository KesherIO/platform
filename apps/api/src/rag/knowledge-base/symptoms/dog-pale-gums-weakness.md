## Pattern Overview

Pale mucous membranes (gums, conjunctiva) combined with weakness or exercise intolerance is a critical presentation indicating significant anemia. This is always a clinical emergency until proven otherwise. The priority is to assess anemia severity, determine whether it is regenerative (active bone marrow response) or non-regenerative (bone marrow failure), and identify the underlying cause. The cause guides both the urgency and the treatment.

This document supports AI-assisted triage and test recommendation. It does not constitute a diagnosis.

## Common Symptoms

- Pale, white, or grayish mucous membranes
- Weakness, exercise intolerance, collapse
- Tachycardia, tachypnea (compensatory)
- Lethargy, reluctance to move
- Icterus (yellow tinge) — suggests hemolytic cause
- Dark red/brown urine (hemoglobinuria) — intravascular hemolysis
- Abdominal distension — possible hemoperitoneum
- Petechiae or bleeding — concurrent thrombocytopenia

## Possible Associated Diseases

| Disease / Pattern                                                | Key Differentiating Feature                                                 |
| ---------------------------------------------------------------- | --------------------------------------------------------------------------- |
| Babesiosis                                                       | Regenerative hemolytic anemia; icterus; tick exposure; tropics/endemic area |
| Immune-mediated hemolytic anemia (IMHA)                          | Autoagglutination; positive Coombs test; spherocytosis on smear             |
| Ehrlichiosis (chronic)                                           | Non-regenerative anemia; thrombocytopenia; hyperglobulinemia                |
| Internal hemorrhage (splenic mass, trauma)                       | Sudden onset; abdominal distension; no icterus; imaging urgent              |
| Hookworm anemia (_Ancylostoma_)                                  | Microcytic hypochromic; puppy or young dog; eosinophilia; fecal positive    |
| Iron-deficiency anemia                                           | Microcytic; chronic blood loss (GI, flea infestation)                       |
| Bone marrow suppression (FeLV in cats — also tick-borne in dogs) | Non-regenerative; pancytopenia                                              |
| Rodenticide toxicity (anticoagulant)                             | Bleeding history; prolonged PT/aPTT; no icterus                             |
| Splenic or hepatic neoplasia with hemoperitoneum                 | Sudden pale gums; distended abdomen; large breed older dog                  |

## Recommended Diagnostic Tests

**Immediate (always):**

- **CBC with differential + reticulocyte count** — defines anemia severity and regenerative status; most critical first step
- **Blood smear** — spherocytes (IMHA), Babesia piroplasms, Heinz bodies, schistocytes

**Guided by CBC result:**

- **Babesia PCR or IFA** — if regenerative hemolytic anemia + icterus + tick exposure
- **Direct Coombs test (DAT)** — if spherocytosis or autoagglutination on smear (suspected IMHA)
- **Serum bilirubin (TBIL)** — hemolytic vs. non-hemolytic origin
- **Fecal flotation** — if microcytic hypochromic anemia in a young dog (hookworms)
- **Coagulation profile (PT, aPTT)** — if bruising, bleeding, or suspected rodenticide toxicity
- **Serum chemistry + urinalysis** — organ involvement, hemoglobinuria
- **Abdominal ultrasound** — mandatory if hemoperitoneum suspected (distended abdomen, sudden onset)
- **Tick-borne disease panel** — if non-regenerative anemia + thrombocytopenia

## When to Prioritize Urgent Evaluation

- **HCT < 15–18%** → emergency transfusion consideration
- Collapse or inability to stand
- Tachycardia (HR > 160 bpm at rest) or tachypnea from anemia
- Pale to white gums (not just pale pink)
- Hemoglobinuria (dark urine) → severe intravascular hemolysis
- Rapidly progressive pallor over hours → active hemorrhage or acute hemolysis
- Abdominal distension + sudden pallor → hemoperitoneum (surgical emergency)
