## Pattern Overview

Anorexia combined with lethargy is the most common presentation in sick cats and, unlike in dogs, carries a specific and serious risk: **hepatic lipidosis**. In overweight or previously obese cats, even 48–72 hours of anorexia can trigger pathological fat mobilization to the liver, leading to hepatic failure if not treated aggressively. Every anorexic cat requires prompt evaluation. Identifying the primary underlying cause is essential because hepatic lipidosis is often secondary to another disease.

This document supports AI-assisted triage and test recommendation. It does not constitute a diagnosis.

## Common Symptoms

- Complete or partial anorexia (days to weeks)
- Lethargy, hiding behavior
- Weight loss
- Icterus (yellow tinge to sclera, skin, mucous membranes)
- Vomiting (intermittent)
- Ptyalism (excessive drooling) — hepatic encephalopathy or nausea
- Dehydration
- Hepatomegaly on palpation

## Possible Associated Diseases

| Disease / Pattern                 | Key Differentiating Feature                                                              |
| --------------------------------- | ---------------------------------------------------------------------------------------- |
| Hepatic lipidosis                 | Overweight cat; > 48–72h anorexia; icterus; elevated ALT + ALP + GGT + TBIL              |
| Chronic kidney disease (CKD)      | Older cat; PU/PD; elevated CREA + BUN; low USG                                           |
| Pancreatitis                      | Difficult to diagnose; elevated fPLI; vomiting; abdominal pain                           |
| GI lymphoma / IBD                 | Chronic vomiting; weight loss; palpable intestinal thickening                            |
| FeLV infection                    | Non-regenerative anemia; pancytopenia; antigen-positive test                             |
| FIV infection                     | Lymphopenia; chronic infections; antibody-positive test                                  |
| Upper respiratory infection (URI) | Nasal congestion reduces smell; anosmia-driven anorexia; discharge                       |
| Dental disease / stomatitis       | Oral pain; halitosis; FIV-associated stomatitis                                          |
| Diabetes mellitus                 | PU/PD; hyperglycemia; glucosuria                                                         |
| Hyperthyroidism                   | Older cat (> 8 years); weight loss despite normal/increased appetite (not pure anorexia) |

## Recommended Diagnostic Tests

**First-line:**

- **CBC with differential** — anemia type, leukocyte pattern
- **Comprehensive serum chemistry panel** — ALT, AST, ALP, GGT, TBIL, BUN, CREA, SDMA, ALB, GLU
- **FeLV antigen + FIV antibody test** — mandatory in all anorexic cats

**Second-line:**

- **Urinalysis with USG** — renal concentrating ability (isosthenuria = CKD risk)
- **Abdominal ultrasound** — liver echogenicity (hyperechoic = lipidosis), biliary tree, kidneys, pancreas, intestines
- **Feline pancreatic lipase (fPLI / Spec fPL)** — most specific test for feline pancreatitis
- **Serum T4** — if cat > 8 years, even if appetite seems reduced; hyperthyroidism can coexist with other diseases
- **Liver fine-needle aspirate + cytology** — if hepatic lipidosis suspected; minimally invasive, usually diagnostic

## When to Prioritize Urgent Evaluation

- Overweight cat with anorexia > 48–72 hours → **hepatic lipidosis risk — evaluate immediately**
- Icterus (yellow sclera or skin) in a cat → urgent evaluation regardless of cause
- Complete anorexia in any cat lasting > 3 days
- Severe lethargy (unable to maintain sternal recumbency)
- Neurological signs (hepatic encephalopathy) — disorientation, head pressing, seizures
- HCT < 20% — severe anemia requiring urgent management
- Suspected FeLV with pancytopenia
